# heart_disease_prediction
Projeto de análise de risco de doença cardíaca coronariana (DCC) em 10 anos, utilizando regressão logística. Este projeto avalia o impacto de variáveis demográficas (sexo, idade) e comportamentais (tabagismo) no risco de DCC, com foco na interpretabilidade dos modelos e identificação de fatores relevantes
